        • java.util.List<java.util.Date> scheduleJob(org.quartz.JobDetail jobDetail, java.util.Collection<? extends org.quartz.Trigger> triggers, boolean sessionRequired, boolean overwriteExisting = false)

          Adds the JobDetail in the Scheduler and schedules the job's triggers.

          throws:
          SchedulerException
          Returns:
          the next fire time for each trigger; the list will be the same size as the number of triggers but individual elements may be null depending on what's returned from the Scheduler.
          Parameters:
          jobDetail - the job data
          triggers - the triggers

        • void scheduleJobs()

          Registers a JobDetail and schedules triggers for each Spring bean that implements SchwartzJob; called at startup and should not be called otherwise.

          throws:
          SchedulerException

        • java.util.Date scheduleTrigger(org.quartz.Trigger trigger, boolean overwriteExisting = false)

          Register the trigger in the Scheduler. If one is found with the same key and overwriteExisting is true, the previous one will be unsheduled and replaced with this one, otherwise the previous will remain scheduled.

          throws:
          SchedulerException
          Returns:
          the next fire date
          Parameters:
          trigger - the trigger
          overwriteExisting - whether to replace existing triggers with the same key
