Package: com.agileorbit.schwartz.util

[Groovy] Class InstantiateInterceptSchedulerFactory

        • com.agileorbit.schwartz.util.InstantiateInterceptSchedulerFactory

        • Intercepts Scheduler creating to get access to the QuartzScheduler and several other helper objects (job store, thread pool, etc.) and configured settings. Not used by default; enable by setting schedulerFactoryClass = InstantiateInterceptSchedulerFactory in SchwartzSchedulerFactory/SchedulerFactoryBean. When in use a QuartzSchedulerObjects instance will be available via QuartzSchedulerObjects.getInstance(scheduler) after the Scheduler is created by the factory bean.
