        • static java.lang.String dhms(long milliseconds)

          Formats milliseconds as days, hours, minutes, seconds, and milliseconds. Always displays minutes, seconds, and milliseconds as XX:XX.XXX, and displays hours if greater than 0 (XX:XX:XX.XXX), and days if greater than 0 (XX:XX:XX:XX.XXX). All sections are padded with 0, so for example 2 hours, 3 minutes, and 14 milliseconds would result in 02:03:014.
