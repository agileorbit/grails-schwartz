Package com.agileorbit.schwartz.util

  • Class Summary 
    Class Description
    InstantiateInterceptSchedulerFactory Intercepts Scheduler creating to get access to the QuartzScheduler and several other helper objects (job store, thread pool, etc.) and configured settings.
    JobSnapshot Used by SchedulerSnapshot to store JobDetails and Triggers.
    QuartzSchedulerObjects Created when using InstantiateInterceptSchedulerFactory; contains the QuartzScheduler created by the factory along with its internal Scheduler and other reachable objects and properties that are otherwise impractical to access.
    SchedulerSnapshot Captures data from several areas of Quartz for use with testing or debugging.
    Utils Utility methods.
