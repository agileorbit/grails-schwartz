Package com.agileorbit.schwartz.builder

  • Class Summary 
    Class Description
    AbstractProperties Abstract base class for all properties classes.
    AbstractSchedulerProperties Abstract base class for all ScheduleBuilder properties classes.
    BuilderFactory Aggregates all properties used to configure the standard Quartz Triggers (SimpleTrigger, CalendarIntervalTrigger, DailyTimeIntervalTrigger, CronTrigger).
    CalendarProperties Properties for configuring the schedule-related values of a CalendarIntervalTrigger using CalendarIntervalScheduleBuilder.
    CommonProperties Manages properties that are shared by other properties classes.
    CronProperties Properties for configuring the schedule-related values of a CronTrigger using CronScheduleBuilder.
    DailyProperties Properties for configuring the schedule-related values of a DailyTimeIntervalTrigger using DailyTimeIntervalScheduleBuilder.
    SimpleProperties Properties for configuring the schedule-related values of a SimpleTrigger using SimpleScheduleBuilder.
    TriggerProperties The other properties classes work with schedule-related values and this one configures the remaining, used to finish building each of the supported trigger types using TriggerBuilder.
