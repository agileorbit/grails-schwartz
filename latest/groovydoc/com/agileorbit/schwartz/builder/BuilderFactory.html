Package: com.agileorbit.schwartz.builder

[Groovy] Class BuilderFactory

    • com.agileorbit.schwartz.builder.BuilderFactory

    • Aggregates all properties used to configure the standard Quartz Triggers (SimpleTrigger, CalendarIntervalTrigger, DailyTimeIntervalTrigger, CronTrigger). Properties can be set using regular setter methods (e.g. void setStartAt(Date)) or Groovy property access (e.g. factory.startAt = ...), and each property also has a variant that sets the value and retuns the factory instance to support method chaining (e.g. BuilderFactory startAt(Date)). Properties can also be set in bulk from a Map (similar to using the Groovy map constructor). The property names are identical to or very similar to those used in the schedule builder classes (CalendarIntervalScheduleBuilder, CronScheduleBuilder, DailyTimeIntervalScheduleBuilder, and SimpleScheduleBuilder) and TriggerBuilder.
