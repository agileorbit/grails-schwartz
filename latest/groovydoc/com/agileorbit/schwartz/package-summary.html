Package com.agileorbit.schwartz

  • Trait Summary 
    Trait Description
    SchwartzJob Implements org.quartz.Job and org.quartz.InterruptableJob and includes methods that define org.quartz.JobDetail and org.quartz.Trigger values to use when registering the class as a job in the org.quartz.Scheduler, along with other utility methods for scheduling and job management.
    StatefulSchwartzJob Extends SchwartzJob and is annotated with the annotations Quartz requires for a job to be considered stateful.
